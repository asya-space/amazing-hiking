# 🌄 Amazing Hiking — Landing Page

<p>This is an older experimental project that I intentionally keep in its original state without further updates.
It was created during a learning phase focused on exploring single-page layouts, navigation patterns, and basic UI interactions.</p>

🧭 The Concept

One of the key experiments in this project is the navigation system:

- A fixed navigation element on the right side of the page
- Section markers (1 / 2 / 3)
- Clicking a number scrolls directly to the corresponding section

 This behavior was implemented using jQuery, as a practical solution for smooth scrolling and section switching.

✨ UI / Visual Experiments

Another focus of the project was visual interaction design:

- custom-styled social media links
- hover and transition effects for interactive feedback
- emphasis on making external links feel integrated into the UI rather than static elements

🛠 Tech Stack
- HTML5
- CSS3
- JavaScript
- jQuery (used for scroll and navigation behavior)
 
⚙️ Note on jQuery
At the time of development, jQuery was used to implement:
- smooth scrolling between sections
- click-based navigation between page blocks

This was a practical choice before switching to more modern vanilla JS approaches.

📌 Project Status

This project is not actively maintained.

It is preserved as a snapshot of early experimentation with:
- single-page layout structure
- UX/navigation patterns
- basic interactive front-end behavior
